# Zotero-jsa

## 1.介绍

Zotero-jsa加载项，一款**学术论文管理软件Zotero** 和**WPS文字**交互插件。

理论支持平台：windows&mac&linux

暂支持平台:Windows

后续等wps官方开放其他平台的宏加载项入口，即可使用
